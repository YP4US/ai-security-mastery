# 🌅 Morning Message — One Line, Every Morning

Open Claude Code in VS Code and send:

```
Good morning. Open my daily card for today.
```

That's it. Claude will open the right `daily-cards/day-NNN.md` file for you.

---

## Then What?

Read the daily card top to bottom (5 minutes).
Work through it in order.
Check boxes as you go.
Close it at night.

**You never need to open any other file during your day.**
The chapters, resources, and templates are reference material — the daily card has everything.

---

## If Claude Opens the Wrong Day

Say: "Today is Day [N]. Open daily-cards/day-0NN.md"

## If Tomorrow's Card Doesn't Exist Yet

It gets created during your evening check-in. If you're starting a day and no card exists, say:
"Generate today's daily card. I'm on Day [N]."
Claude will build it from the chapter files and template.
