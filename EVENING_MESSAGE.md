# 🌙 Evening Message — Every Evening at 22:45

Open Claude Code in VS Code and send:

```
Evening check-in. Day [N]. Ready.
```

---

## The 7 Questions (One at a Time — Be Honest)

**1.** "Startup block: what did you work on for Mediqrate or PracticeCLB?"

**2.** "Governance warm-up: what Track A topic did you cover?"

**3.** "Deep work: what did you complete on Track B?"

**4.** "Karpathy: which video(s)? One security insight?"

**5.** "What didn't you finish today and why?"

**6.** "Morale 1–10 and one sentence why?"

**7.** "Any blockers for tomorrow?"

---

## What Claude Does After (Automatically)

1. Creates `daily-logs/YYYY-MM-DD-day-NNN.md` from your answers
2. Updates `PROGRESS.md`
3. Updates `resources/karpathy-watch-log.md`
4. **Creates tomorrow's daily card:** `daily-cards/day-{N+1}.md`
5. Commits and pushes everything

You wake up to a ready daily card. No planning needed.
