**Step 1: Python environment (15 min)**
```bash
# Check Python version — needs to be 3.10 or higher
python3 --version

# Create a virtual environment for security tools
python3 -m venv ~/security-env

# Activate it (you'll do this every session)
source ~/security-env/bin/activate    # Mac or Linux
# security-env\Scripts\activate       # Windows
```
- [ ] Python 3.10+ confirmed
- [ ] Virtual environment created and activated

**Step 2: Install garak — your main LLM scanner (20 min)**
```bash
# Make sure your environment is activated first
source ~/security-env/bin/activate

pip install garak
pip install requests rich python-dotenv
pip install openai anthropic

# Verify garak installed correctly:
garak --help
```
- [ ] garak installed (`garak --help` shows output)

**Step 3: Install Ollama — free local LLM, no API cost (20 min)**
```
Download from: https://ollama.ai/download
Install it like a normal app, then open it.

After it's running, open your terminal and run:
```
```bash
# Download a small model (about 600MB):
ollama pull llama3.2:1b

# Test it's working:
ollama run llama3.2:1b "Say hello"
# It should respond. Takes 1-2 min first time.
```
- [ ] Ollama installed
- [ ] llama3.2:1b downloaded and responding
